Le scrapping se fait grace a une classe appelée InstagramBot.
Elle comporte 05 méthodes
    -signIn : pour entrer les identifiants de connexion
    -scrapeWithHashTag : pour se rendre sur la page du hastTag donné en parametre
    -Scrape : pour récupérer tous les postes de la pages
    -ScrapePosts : pour scrapper un post spécifique.
